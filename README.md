# Simple weather APP

## Setup

1. `npm i`
2. create `.env` file (look `.env.example` as sample)
3. `npm run dev`

## Libraries 

- react
- react-dom
- redux-toolkit with rtk query
- react-redux
- redux-persist
- sass
- moment
- formik
- classnames
- react-modal
- react-loader-spinner
- react-toastify
- prettier

## API

- openweathermap - https://openweathermap.org/api
